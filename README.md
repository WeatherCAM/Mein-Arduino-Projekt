# Mein Arduino Projekt

Dies ist ein Beispielprojekt für den Arduino. Es steuert eine LED basierend auf der Helligkeit eines Lichtsensors.

## Installation

1. Installiere die Bibliothek `Adafruit_Sensor` über den Arduino Library Manager.
2. Lade die Datei `meinprojekt.ino` auf dein Arduino.

## Benötigte Bibliotheken

- Adafruit_Sensor
- BH1750

## Verwendung

Schließe einen BH1750-Lichtsensor an das Arduino an. Der Sketch wird die Helligkeit messen und eine LED ein- oder ausschalten, abhängig von der gemessenen Helligkeit.
